{% trans "1. Objet" %}

{% blocktrans trimmed %}Les présentes Conditions de Partenariat définissent le cadre général applicable à toute collaboration entre Revival et toute personne physique ou morale, ci-après désignée comme le Partenaire, dans le cadre de projets digitaux, commerciaux, technologiques ou stratégiques.{% endblocktrans %}

{% blocktrans trimmed %}Elles ont pour finalité d’établir une relation claire, professionnelle et équilibrée, orientée vers la confiance, la performance et la croissance mutuelle.{% endblocktrans %}

{% trans "2. Domaines de collaboration" %}

{% blocktrans trimmed %}Revival peut développer des partenariats dans plusieurs domaines, selon les besoins du marché et le profil du partenaire.{% endblocktrans %}

  • {% trans "Développement d’applications et de solutions digitales." %}
  • {% trans "Distribution, commercialisation ou revente de services et produits." %}
  • {% trans "Apport d’affaires, génération d’opportunités et acquisition clients." %}
  • {% trans "Intégration technologique, innovation et développement conjoint." %}
  • {% trans "Marketing, communication, visibilité et actions de marque." %}
  • {% trans "Formation, accompagnement, conseil ou déploiement d’expertise." %}

{% trans "3. Engagements de Revival" %}

  • {% blocktrans trimmed %}Fournir des solutions professionnelles, performantes et adaptées aux réalités du marché africain.{% endblocktrans %}
  • {% trans "Assurer un accompagnement technique et stratégique aligné sur les objectifs du partenariat." %}
  • {% trans "Protéger la confidentialité des informations échangées." %}
  • {% trans "Respecter les termes convenus dans les accords spécifiques conclus avec le partenaire." %}
  • {% blocktrans trimmed %}Mettre à disposition, lorsque cela est prévu, les outils, supports ou ressources utiles à la collaboration.{% endblocktrans %}

{% trans "4. Engagements du Partenaire" %}

  • {% trans "Respecter les valeurs, l’image, les standards de qualité et les intérêts légitimes de Revival." %}
  • {% trans "Fournir des informations exactes, à jour et exploitables pour la bonne conduite du partenariat." %}
  • {% trans "Collaborer avec professionnalisme, transparence et bonne foi." %}
  • {% blocktrans trimmed %}S’abstenir de tout comportement susceptible de porter atteinte à Revival, à ses clients ou à ses activités.{% endblocktrans %}
  • {% trans "Honorer les obligations contractuelles, financières ou opérationnelles qui lui incombent." %}

{% trans "5. Modèles de partenariat" %}

{% blocktrans trimmed %}Revival peut proposer différents formats de collaboration, chacun pouvant donner lieu à des conditions particulières.{% endblocktrans %}

  • {% blocktrans trimmed %}Partenariat commercial : le partenaire intervient comme apporteur d’affaires, distributeur ou revendeur et peut percevoir une commission selon les ventes réalisées.{% endblocktrans %}
  • {% blocktrans trimmed %}Partenariat stratégique : collaboration de moyen ou long terme autour de projets communs, d’un marché, d’un produit ou d’un écosystème partagé.{% endblocktrans %}
  • {% blocktrans trimmed %}Partenariat technologique : intégration de solutions, co-développement, interconnexion technique ou contribution d’expertise.{% endblocktrans %}
  • {% blocktrans trimmed %}Partenariat institutionnel : coopération avec des organisations, entreprises, institutions publiques ou privées.{% endblocktrans %}

{% blocktrans trimmed %}Chaque modèle peut être complété par un accord spécifique détaillant les obligations opérationnelles, commerciales ou financières propres à la collaboration concernée.{% endblocktrans %}

{% trans "6. Rémunération et commissions" %}

{% blocktrans trimmed %}Les modalités de rémunération dépendent de la nature du partenariat et sont définies dans les accords applicables.{% endblocktrans %}

  • {% trans "Commissions sur ventes ou sur projets apportés." %}
  • {% trans "Partage de revenus ou répartition contractuelle des gains." %}
  • {% trans "Honoraires fixes ou forfaits de prestation." %}
  • {% trans "Bonus de performance, selon les objectifs convenus." %}

{% blocktrans trimmed %}Les paiements sont effectués selon les délais, justificatifs et conditions prévus dans l’accord spécifique conclu entre les parties.{% endblocktrans %}

{% trans "7. Confidentialité" %}

{% blocktrans trimmed %}Toutes les informations techniques, stratégiques, commerciales, financières ou opérationnelles échangées dans le cadre du partenariat sont confidentielles, sauf mention contraire écrite.{% endblocktrans %}

{% blocktrans trimmed %}Le Partenaire s’engage à ne pas divulguer, reproduire, exploiter ou transmettre ces informations à un tiers sans autorisation écrite préalable de Revival.{% endblocktrans %}

{% trans "8. Propriété intellectuelle" %}

{% blocktrans trimmed %}Sauf disposition contraire convenue par écrit, les solutions, technologies, marques, méthodologies, contenus, visuels, documents et éléments créés ou exploités par Revival demeurent sa propriété exclusive.{% endblocktrans %}

{% blocktrans trimmed %}Le Partenaire ne peut utiliser les éléments appartenant à Revival que dans le cadre strict du partenariat et selon les autorisations accordées.{% endblocktrans %}

{% trans "9. Durée et résiliation" %}

{% blocktrans trimmed %}Le partenariat peut être conclu pour une durée déterminée ou indéterminée, selon les termes fixés entre les parties.{% endblocktrans %}

  • {% trans "Chaque partie peut y mettre fin en cas de non-respect des engagements convenus." %}
  • {% blocktrans trimmed %}Une résiliation peut également intervenir en cas de comportement préjudiciable, de force majeure ou de motif légitime prévu par l’accord applicable.{% endblocktrans %}
  • {% blocktrans trimmed %}Sauf faute grave ou urgence particulière, une notification préalable doit être adressée à l’autre partie.{% endblocktrans %}

{% trans "10. Responsabilités" %}

{% blocktrans trimmed %}Chaque partie demeure responsable de ses actes, déclarations, engagements et obligations dans le cadre du partenariat.{% endblocktrans %}

{% blocktrans trimmed %}Revival ne saurait être tenu responsable des dommages, pertes ou conséquences résultant d’une mauvaise utilisation de ses services, d’une information erronée fournie par le partenaire ou d’un manquement imputable à ce dernier.{% endblocktrans %}

{% trans "11. Conformité légale" %}

{% blocktrans trimmed %}Le Partenaire s’engage à respecter les lois, règlements et obligations applicables à son activité, notamment en matière de commerce, fiscalité, concurrence, propriété intellectuelle et protection des données.{% endblocktrans %}

{% trans "12. Modification des conditions" %}

{% blocktrans trimmed %}Revival se réserve le droit de mettre à jour les présentes conditions afin de tenir compte de l’évolution de ses activités, de ses offres ou du cadre réglementaire applicable.{% endblocktrans %}

{% trans "Les partenaires seront informés de toute modification importante par les canaux appropriés." %}

{% trans "13. Droit applicable et juridiction" %}

{% blocktrans trimmed %}Les présentes conditions sont régies par les lois applicables en République Démocratique du Congo et, le cas échéant, par les principes pertinents du droit OHADA lorsque ceux-ci trouvent à s’appliquer.{% endblocktrans %}

{% blocktrans trimmed %}Tout litige relatif à leur interprétation, exécution ou validité relève des juridictions compétentes, sauf mécanisme particulier prévu dans un accord spécifique conclu entre les parties.{% endblocktrans %}
